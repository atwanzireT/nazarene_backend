Jonahs Alumni API Documentation

Comprehensive reference for all major and minor endpoints.

GET /api/users/GET

Admin only: View all users. Filter by role, approval, search, and order.

POST /api/account-application/POST

Submit an alumni account application with full profile details.

GET /api/applications/GET

Admin only: View submitted applications with filters (approved, year joined/left).

GET /api/projects/GET

View all projects with progress, activities, dates, and active filters.

GET /api/project-images/GET

List images associated with projects.

GET /api/activities/GET

View project-related activities, filterable by project ID.

GET /api/activity-images/GET

Fetch activity gallery photos with captions.

GET /api/events/GET

List all events, with upcoming/past filters and participation tracking.

GET /api/event-images/GET

Get associated images for each event.

POST /api/registrations/POST

Register authenticated user for an event. Prevent duplicate or expired registrations.

GET /api/registrations/GET

Get list of registrations for current authenticated user.

GET /api/notifications/GET

Fetch user notifications with read status and time ago metadata.
