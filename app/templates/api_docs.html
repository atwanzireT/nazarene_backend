Documentation for the Alumni Association Management System

POST /api/account-application/ Post

Submit a new account application. This is open to all potential alumni members.

Request Body

Field Type Required Description
full_first_name string Yes Applicant's first name
full_surname string Yes Applicant's surname
email string Yes Unique email address
mobile_phone_1 string Yes Primary mobile number
year_joined_jonahs integer Yes Year joined Jonahs
year_left_jonahs integer Yes Year left Jonahs
consent_given boolean Yes Must be true to submit

Response

Success: 201 Created

{
    "message": "Application submitted successfully"
}
GET /api/users/ Get

List all users. Requires admin privileges.

Filter Options:

?is_approved=true - Filter by approval status
?role=ADMIN - Filter by role
?search=john - Search by name/email
?ordering=-date_joined - Sort by date

Response

Success: 200 OK

[
    {
        "id": 1,
        "username": "admin",
        "full_name": "Admin User",
        "email": "admin@example.com",
        "phone_number": "+1234567890",
        "is_approved": true,
        "role": "ADMIN",
        "role_display": "Admin",
        "date_joined": "2023-01-01T00:00:00Z"
    }
]
